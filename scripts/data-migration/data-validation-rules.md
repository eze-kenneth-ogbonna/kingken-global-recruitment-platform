# Data Validation Rules

**Platform:** Kingken Global Recruitment Platform  
**Company:** Kingken Global Travel Agency Ltd.  
**Sheet:** Master Data

---

## Overview

These rules define the accepted format and allowed values for every field in the Master Data sheet. Violations should be flagged during data entry, migration, and automated processing.

---

## Field Validation Rules

### CandidateID (Column A)

| Rule | Detail |
|---|---|
| Format | `KENG-YYYYMMDD-NNNN` where YYYYMMDD = date, NNNN = 4-digit zero-padded number |
| Example | `KENG-20241215-0042` |
| Must be unique | No two rows may share the same CandidateID |
| Not empty | Required on all rows |
| Auto-generated | Must not be manually typed (generated by Apps Script or Code by Zapier) |

**Regex pattern:** `^KENG-\d{8}-\d{4}$`

**Google Sheets validation formula (paste in Data → Data validation → Custom formula):**
```
=AND(LEN(A2)=18, LEFT(A2,5)="KENG-", ISNUMBER(VALUE(MID(A2,6,8))), MID(A2,14,1)="-", ISNUMBER(VALUE(RIGHT(A2,4))))
```

---

### FullName (Column B)

| Rule | Detail |
|---|---|
| Not empty | Required |
| Minimum length | 3 characters |
| Allowed characters | Letters (A–Z, a–z), spaces, hyphens (`-`), apostrophes (`'`) |
| Disallowed | Numbers, special characters (! @ # $ % etc.) |
| Maximum length | 100 characters |

**Google Sheets validation formula:**
```
=AND(LEN(TRIM(B2))>=3, REGEXMATCH(B2, "^[A-Za-z\s\-']+$"))
```

---

### Timestamp (Column C)

| Rule | Detail |
|---|---|
| Format | Date/time — auto-populated by form or Zapier |
| Not empty | Required |
| Valid date | Must be a recognisable date value (not text) |

**Google Sheets validation:** Set column C format to **Date time** (Format → Number → Date time).

---

### Phone (Column D)

| Rule | Detail |
|---|---|
| Format | E.164 — starts with `+`, followed by 10–15 digits |
| Example | `+2348012345678` |
| Must include country code | Numbers without `+` prefix are invalid after normalisation |
| Minimum length | 11 characters (including `+`) |
| Maximum length | 16 characters (including `+`) |
| Disallowed characters | Spaces, dashes, parentheses (all removed during normalisation) |

**Google Sheets validation formula:**
```
=AND(LEFT(D2,1)="+", LEN(D2)>=11, LEN(D2)<=16, ISNUMBER(VALUE(MID(D2,2,LEN(D2)-1))))
```

---

### Country (Column E)

| Rule | Detail |
|---|---|
| Allowed values | Must be from the approved country list (see below) |
| Case | Title case preferred (e.g. `Nigeria` not `NIGERIA` or `nigeria`) |
| Not empty | Required |

**Approved country list:**

```
Nigeria, Ghana, Kenya, Ethiopia, Uganda, Tanzania, Cameroon, Senegal,
Kuwait, UAE, Qatar, Saudi Arabia, Bahrain, Oman, Egypt
```

**Google Sheets validation (dropdown):**
1. Select the entire column E (from row 2 down).
2. Data → Data validation → Criteria: **List of items**.
3. Paste: `Nigeria,Ghana,Kenya,Ethiopia,Uganda,Tanzania,Cameroon,Senegal,Kuwait,UAE,Qatar,Saudi Arabia,Bahrain,Oman,Egypt`
4. ☑ Show warning / ☑ Reject input.

**Google Sheets custom formula:**
```
=OR(E2="Nigeria",E2="Ghana",E2="Kenya",E2="Ethiopia",E2="Uganda",E2="Tanzania",E2="Cameroon",E2="Senegal",E2="Kuwait",E2="UAE",E2="Qatar",E2="Saudi Arabia",E2="Bahrain",E2="Oman",E2="Egypt")
```

| Rule | Detail |
|---|---|

---

### AI_Score (Column L)

| Rule | Detail |
|---|---|
| Type | Numeric integer |
| Minimum | 0 |
| Maximum | 100 |
| Empty allowed | Yes — empty means not yet scored |
| Special value | `SCORE_ERROR` — indicates a failed API call (review manually) |

**Google Sheets validation (allow empty):**
```
=OR(L2="", L2="SCORE_ERROR", AND(ISNUMBER(L2), L2>=0, L2<=100))
```

## Apps Script Validation Function

The following function can be called during data import or after a batch migration to flag rows that violate any of the above rules.

```javascript
/**
 * @file data-validation-rules (reference only — paste into Apps Script)
 * @description Validates all rows in Master Data against defined business rules.
 *              Writes validation errors to a "Validation Errors" column.
 * @company Kingken Global Travel Agency Ltd.
 */

var VALID_COUNTRIES  = ['Nigeria','Ghana','Kenya','Ethiopia','Uganda','Tanzania','Cameroon','Senegal','Kuwait','UAE','Qatar','Saudi Arabia','Bahrain','Oman','Egypt'];
var VALID_POSITIONS  = ['Domestic Worker','Driver','Cleaner','Cook','Security Guard','Construction Worker','Nanny/Caregiver','Office Cleaner','Factory Worker','Other'];
var VALID_STATUSES   = ['New','Screened','Approved','Rejected','Processing','Deployed'];
var VALID_PASSPORT   = ['Yes','No'];
var CANDIDATE_ID_RE  = /^KENG-\d{8}-\d{4}$/;
var PHONE_RE         = /^\+\d{10,15}$/;
var NAME_RE          = /^[A-Za-z\s\-']{3,100}$/;

/**
 * Validates all rows in Master Data and writes errors to a Validation Errors column.
 *
 * @return {void}
 */
function validateAllRows() {
  var ss     = SpreadsheetApp.getActiveSpreadsheet();
  var sheet  = ss.getSheetByName('Master Data');
  if (!sheet) { Logger.log('Sheet not found'); return; }

  var lastRow = sheet.getLastRow();
  if (lastRow < 2) { Logger.log('No data rows'); return; }

  var data       = sheet.getRange(2, 1, lastRow - 1, 12).getValues();
  var errorCol   = _getOrCreateColumn(sheet, 'ValidationErrors');
  var errorValues = [];
  var errorCount  = 0;

  data.forEach(function (row, i) {
    var errors = [];

    var candidateId = String(row[0]).trim();
    var fullName    = String(row[1]).trim();
    var phone       = String(row[3]).trim();
    var country     = String(row[4]).trim();
    var position    = String(row[5]).trim();
    var experience  = row[6];
    var passport    = String(row[7]).trim();
    var status      = String(row[10]).trim();
    var aiScore     = row[11];

    if (!CANDIDATE_ID_RE.test(candidateId)) errors.push('Invalid CandidateID format');
    if (!NAME_RE.test(fullName))            errors.push('FullName missing or invalid characters');
    if (!PHONE_RE.test(phone))              errors.push('Phone not in E.164 format (+XXXXXXXXXXX)');
    if (VALID_COUNTRIES.indexOf(country) === -1)  errors.push('Country not in approved list: ' + country);
    if (VALID_POSITIONS.indexOf(position) === -1) errors.push('Position not in approved list: ' + position);
    if (isNaN(experience) || experience < 0 || experience > 40) errors.push('ExperienceYears must be 0–40');
    if (VALID_PASSPORT.indexOf(passport) === -1) errors.push('PassportAvailable must be Yes or No');
    if (status && VALID_STATUSES.indexOf(status) === -1) errors.push('Invalid Status: ' + status);
    if (aiScore !== '' && aiScore !== 'SCORE_ERROR' && (isNaN(aiScore) || aiScore < 0 || aiScore > 100)) {
      errors.push('AI_Score must be 0–100 or empty');
    }

    if (errors.length > 0) errorCount++;
    errorValues.push([errors.join(' | ')]);
  });

  sheet.getRange(2, errorCol, errorValues.length, 1).setValues(errorValues);

  SpreadsheetApp.getUi().alert(
    '✅ Validation Complete\n\n' +
    'Rows checked : ' + data.length + '\n' +
    'Rows with errors: ' + errorCount + '\n\n' +
    'See "ValidationErrors" column for details.'
  );
}

/**
 * Gets or creates a column with the given header name.
 *
 * @private
 * @param  {GoogleAppsScript.Spreadsheet.Sheet} sheet  Target sheet.
 * @param  {string}                             header Column header text.
 * @return {number} 1-based column index.
 */
function _getOrCreateColumn(sheet, header) {
  var lastCol = sheet.getLastColumn();
  var headers = sheet.getRange(1, 1, 1, lastCol).getValues()[0];
  for (var c = 0; c < headers.length; c++) {
    if (String(headers[c]).trim() === header) return c + 1;
  }
  var newCol = lastCol + 1;
  sheet.getRange(1, newCol).setValue(header);
  return newCol;
}
```

Use this table during manual spot-checks:

| Column | Rule | Pass / Fail |
|---|---|---|
| A — CandidateID | Format `KENG-YYYYMMDD-NNNN`, unique | |
| B — FullName | Not empty, letters/spaces/hyphens only, 3+ chars | |
| C — Timestamp | Valid date value, not empty | |
| D — Phone | Starts with `+`, 11–16 chars, digits only after `+` | |
| E — Country | From approved list of 15 countries | |
| F — Position | From approved list of 10 categories | |
| G — ExperienceYears | Numeric, 0–40 | |
| H — PassportAvailable | `Yes` or `No` only | |
| K — Status | One of 6 approved values | |
| L — AI_Score | Numeric 0–100, or empty, or `SCORE_ERROR` | |
