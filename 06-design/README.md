# Design System

This directory contains UI/UX and design system documentation.

## Structure

* **components/** - Component specifications and guidelines
* **patterns/** - Design patterns and best practices
* **assets/** - Design assets, icons, and resources
* **style-guide/** - Visual design guidelines

## Overview

The design system includes:
- Component library documentation
- Design tokens and theming
- UI/UX guidelines and principles
- Accessibility standards
- Responsive design patterns
