# CRM System

This directory contains the Customer Relationship Management (CRM) system components.

## Structure

* **schemas/** - Database schemas and data models
* **workflows/** - Business process workflows and automation
* **api/** - CRM-specific API endpoints and services

## Overview

The CRM system manages:
- Customer data and profiles
- Candidate relationships
- Communication tracking
- Pipeline management
- Recruitment workflows
