# Job Requests Sheet – Column Documentation

**Sheet Tab Name:** `Job Requests`
**Purpose:** Tracks all worker placement requests received from employers.

---

## Column Reference (A–M)

| Col | Field Name | Type | Description |
|-----|-----------|------|-------------|
| A | Job ID | Text | Unique job request identifier |
| B | Employer ID | Text | References EmployerID from Employers sheet |
| C | Employer Name | Formula | Auto-populated via VLOOKUP from Employers sheet |
| D | Job Title / Position | Text | The job role being filled |
| E | Workers Required | Number | Number of workers needed for this request |
| F | Salary USD | Number | Offered monthly salary in USD |
| G | Location / Country | Dropdown | Country where workers will be deployed |
| H | Contract Duration | Number | Contract length in months |
| I | Date Submitted | Date | Date the job request was received |
| J | Required Start Date | Date | Desired deployment start date |
| K | Status | Dropdown | Current status of this job request |
| L | Assigned Recruiter | Text | Name of the recruiter handling this request |
| M | Notes | Text | Additional notes and special requirements |

---

## Column A – Job ID

**Format:** `JOB-YYYYMMDD-NNNN`

**Examples:**
- `JOB-20260115-0001` (first job request on 15 Jan 2026)
- `JOB-20260301-0012` (twelfth job request on 1 Mar 2026)

**Rules:**
- YYYYMMDD = date the job request was received / created
- NNNN = sequential number for that day, padded to 4 digits
- Assigned manually or auto-generated by Zapier
- Never reuse or modify a Job ID once assigned

---

## Column B – Employer ID

**Type:** Text (references Employers!A)  
**Required:** Yes  
**Example:** `EMP-20260115-0001`  
**Note:** Must exactly match an EmployerID in the Employers sheet for the VLOOKUP in column C to work.

---

## Column C – Employer Name

**Type:** Formula  
**Required:** Auto-populated (do not type manually)  
**Formula (row 2):**
```
=IFERROR(VLOOKUP(B2,Employers!A:B,2,FALSE),"")
```

**Formula Explanation:**
- `VLOOKUP(B2, ...)` → looks up the value in B2 (Employer ID)
- `Employers!A:B` → searches in the Employers sheet, columns A through B
- `2` → returns the value from column 2 (B = Company Name)
- `FALSE` → exact match required
- `IFERROR(...,"")` → shows blank if Employer ID is not found (prevents #N/A errors)

**To apply:** Enter formula in C2, then drag down to C1000.

## VLOOKUP Explanation (Column C)

The VLOOKUP formula `=IFERROR(VLOOKUP(B2,Employers!A:B,2,FALSE),"")` works as follows:

1. **Search value:** `B2` — the Employer ID entered in column B
2. **Search range:** `Employers!A:B` — looks across columns A and B of the Employers sheet
3. **Column index:** `2` — returns the value from the 2nd column of the range (column B = Company Name)
4. **Exact match:** `FALSE` ensures the ID must match exactly
5. **IFERROR wrapper:** Returns `""` (empty) instead of `#N/A` when the ID isn't found

**Important:** The formula only works if:
- The Employer sheet tab is named exactly `Employers` (capital E, no spaces)
- The Employer ID in column B exactly matches column A of the Employers sheet

## How Zapier Creates Rows Here from Employer Form

### Zap: Employer Request Form → Job Requests Sheet

When an employer submits the Employer Request Form requesting workers, Zapier creates a new row in this sheet.

**Trigger:** New Google Forms response (Employer Request Form)  
**Action:** Google Sheets → Create Spreadsheet Row → Job Requests sheet

**Field Mapping:**

| Form Field | Sheet Column |
|-----------|-------------|
| Auto-generated | A (Job ID) via Formatter: `JOB-{{date:YYYYMMDD}}-{{zap_meta_human_id}}` |
| Employer ID (from previous Zap that created Employer) | B (Employer ID) |
| (Auto via VLOOKUP) | C (Employer Name) — formula already in sheet |
| Job Title / Role Needed | D (Job Title) |
| Number of Workers | E (Workers Required) |
| Offered Salary | F (Salary USD) |
| Deployment Country | G (Location/Country) |
| Contract Duration | H (Contract Duration) |
| Today's date | I (Date Submitted) |
| Preferred Start Date | J (Required Start Date) |
| (Hardcoded) | K (Status) = "Open" |
| (Auto-assign logic) | L (Assigned Recruiter) |

### Zapier Steps:
1. Trigger: Google Forms → New Response → Employer Request Form
2. Action 1: Formatter → Date → format today as YYYYMMDD
3. Action 2: Formatter → Number → pad sequential counter to 4 digits
4. Action 3: Google Sheets → Create Row → Job Requests sheet → map fields

## How Recruiter Assignment Works

**Manual Assignment:**
- COO or Operations Manager reviews new Open jobs each morning
- Assigns a recruiter based on: country expertise, current workload, industry knowledge
- Updates column L with recruiter name
- Sends WhatsApp notification to recruiter

**Automated Assignment via Zapier (optional):**
- Set up Zapier with Google Sheets → New Row trigger
- Action: Send WhatsApp via WATI to default recruiter notifying of new job
- Recruiter then accepts and COO confirms

**Recruiter Notification Template:**
```
New Job Request Assigned to You:
Job ID: [Job ID]
Employer: [Company Name]
Position: [Job Title]
Workers Needed: [Number]
Country: [Country]
Start Date: [Required Start Date]
Please begin sourcing candidates immediately.
```

## Sheet Maintenance

- **Review frequency:** Daily (COO reviews Open and Matching jobs)
- **Archiving:** Move Closed/Cancelled rows to "Job Requests – Archive" after 6 months
- **Linked sheets:** Employers (via VLOOKUP), Pipeline (references Job ID)
