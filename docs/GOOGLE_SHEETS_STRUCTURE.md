## Overview

The platform uses a single Google Sheets workbook with **7 tabs** acting as a lightweight CRM and ATS (Applicant Tracking System). All automation flows read from and write to this spreadsheet.

| Tab # | Tab Name | Row Count (Est.) | Primary Key |
|-------|----------|------------------|-------------|
| 1 | Master Data | Up to 10,000 | CandidateID |
| 2 | Employers | Up to 1,000 | EmployerID |
| 3 | Job Requests | Up to 500 | JobID |
| 4 | Pipeline | Up to 10,000 | PipelineID |
| 5 | Deals | Up to 2,000 | DealID |
| 6 | Tasks | Up to 5,000 | TaskID |
| 7 | Dashboard | ~30 rows | — |

## Tab 1: Master Data

**Purpose:** Central repository for all candidate/worker records. Auto-populated by Zap 01 (Google Forms) and Zap 02 (WhatsApp bot).

| Col | Header | Data Type | Example Value | Validation Rules |
|-----|--------|-----------|---------------|-----------------|
| A | CandidateID | Text (Auto) | `CAND-0001` | Auto-generated by Zapier formula: `"CAND-"&TEXT(ROW()-1,"0000")` |
| B | Timestamp | DateTime | `2024-01-15 09:32:00` | Auto from form submission |
| C | Full Name | Text | `Amara Osei` | Min 2 chars, required |
| D | Phone | Text | `+2348012345678` | Include country code |
| E | Email | Email | `amara@gmail.com` | Valid email format or blank |
| F | Country | Dropdown | `Nigeria` | Nigeria, Ghana, Kenya, Ethiopia, Uganda, Tanzania, Cameroon, Senegal, Other |
| G | Position | Dropdown | `Domestic Worker` | Domestic Worker, Driver, Security Guard, Cleaner, Nanny, Caregiver, Cook, Office Assistant, Labourer, Other |
| H | Experience | Dropdown | `3–5 years` | Less than 1 year, 1–2 years, 3–5 years, 5+ years |
| I | Passport | Dropdown | `Yes — Valid` | Yes — Valid, Yes — Expired, No — Not Yet |
| J | CV_Link | URL | `https://drive.google.com/...` | Google Drive link to uploaded CV |
| K | Status | Dropdown | `New` | New, Screened, Approved, Processing, Visa Applied, Deployed, Complete, Rejected |
| L | AI_Score | Number (0–100) | `78` | Integer, 0–100 range, written by Zapier/OpenAI |

### Status Value Definitions

| Status | Meaning | Next Action |
|--------|---------|-------------|
| New | Just submitted, not yet reviewed | AI scores; recruiter reviews within 24h |
| Screened | Recruiter has reviewed; needs approval | Head of Recruitment to approve |
| Approved | Approved for deployment consideration | Match to job request |
| Processing | Documentation in progress | Collect passport, medicals |
| Visa Applied | Visa application submitted | Wait for embassy |
| Deployed | Worker has traveled and started job | Invoice employer |
| Complete | Placement fully closed, fee paid | Archive |
| Rejected | Does not meet criteria | Send polite decline message |

## Auto-ID Generation (Google Apps Script)

To auto-generate IDs for each new row, add this script via **Extensions → Apps Script**:

```javascript
function onEdit(e) {
  const sheet = e.range.getSheet();
  const row = e.range.getRow();
  const col = e.range.getColumn();

  // Master Data: auto-generate CandidateID in column A
  if (sheet.getName() === 'Master Data' && col === 3 && row > 1) {
    const idCell = sheet.getRange(row, 1);
    if (idCell.getValue() === '') {
      idCell.setValue('CAND-' + String(row - 1).padStart(4, '0'));
    }
  }

  // Employers: auto-generate EmployerID in column A
  if (sheet.getName() === 'Employers' && col === 3 && row > 1) {
    const idCell = sheet.getRange(row, 1);
    if (idCell.getValue() === '') {
      idCell.setValue('EMP-' + String(row - 1).padStart(4, '0'));
    }
  }

  // Deals: auto-generate DealID in column A
  if (sheet.getName() === 'Deals' && col === 2 && row > 1) {
    const idCell = sheet.getRange(row, 1);
    if (idCell.getValue() === '') {
      idCell.setValue('DEAL-' + String(row - 1).padStart(4, '0'));
    }
  }
}
```

Save and authorize the script. IDs will be generated automatically as new rows are added.
