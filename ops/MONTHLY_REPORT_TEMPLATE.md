# Monthly Report Template — Kingken Global Recruitment Platform

**Document Type:** Monthly Business Performance Report
**Frequency:** First Monday of each month (covering the previous calendar month)
**Owner:** COO prepares with input from all department heads
**Distribution:** CEO, Board / Investors (if applicable), All Senior Staff
**Company:** Kingken Global Travel Agency Ltd.

## Report Format

This template is used to produce the monthly business performance report. Data is sourced from Google Sheets Dashboard, Zapier task history, and WATI analytics. Archive a PDF copy in Google Drive after each report.
