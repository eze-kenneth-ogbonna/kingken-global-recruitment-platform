# AI System

This directory contains the AI automation and intelligence layer components.

## Structure

* **models/** - AI/ML models and configurations
* **automation/** - Automated workflows and decision engines
* **integrations/** - Third-party AI service integrations

## Overview

The AI system provides:
- Intelligent candidate matching
- Resume parsing and analysis
- Automated screening and recommendations
- Predictive analytics
- Natural language processing capabilities
